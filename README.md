# Pizzeria FrontEnd

Strona internetowa imitująca pizzerie.
Funkcje:
- wyświetla produkty z bazy danych
- rejestracje i logowanie użytkownika
- złożenie zamówienia przez użytkownika
- wyświetlenie histori zamówień
- autoryzacje za pomocą JWT
